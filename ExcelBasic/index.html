Excel Basic

พื้นฐาน Microsoft Excel

เทคนิคการใช้งาน Microsoft Excel ระดับพื้นฐาน

เนื้อหา

Interface ตาราง การจัดการตาราง การดำเนินการทางคณิตศาสตร์ ตัวดำเนินการเปรียบเทียบ ฟังก์ชัน ฟังก์ชัน IF ฟังก์ชัน VLOOKUP การสร้างกราฟ เทคนิคเพิ่มเติม Workshop

โหมดกลางคืน สมาชิกกลุ่ม

Microsoft Excel คืออะไร ?

คือโปรแกรมตัวนึงในชุดโปรแกรม Microsoft Office มี ความสามารถ ด้านการวิเคราะห์ คำนวณ และการจัดการข้อมูลในรูปแบบตารางที่เรียกว่า Spreadsheet และสามารถนำไปต่อยอดเป็นโปรแกรมคำนวนแบบทั่วไป

ควรมีความรู้เกี่ยวกับ Excel มากแค่ไหน ?

ปัจจุบันในการทำงานต่างๆ แถบทุกสายงานก็ต้องมีการใช้โปรแกรม Microsoft Excel แน่นอน ซึ่งไว้ใช้สำหรับการทำบัญชี คำนวนทางตัวเลขและสรุปออกมาเป็นสารสนเทศที่จะเอาไปใช้งานต่อไปซึ่งข้อมูลต่างๆ จะต้องผ่านตัวกลางที่เรียกกันว่า “สูตร” ที่สร้างขึ้นบน Cell ต่างๆใน Excel เพราะฉะนั้น ถ้าเข้าใจถึงเรื่องการสร้างสูตร ก็เพียงพอสำหรับความรู้ในการใช้ Excel ที่จะเอาไปต่อยอดไปทำงานจริงได้

ถ้าเรารู้วิธีสร้างสูตรแล้วควรจะเรียนรู้อะไรต่อไปใน Excel ?

ควรศึกษาฟังก์ชันพื้นฐานต่างๆที่ติดมากับโปรแกรม Microsoft Excel ให้เข้าใจว่าเราควรจะใช้งานตอนไหนถึงจะเหมาะสม ซึ่งมีหลายฟังก์ชันที่ทำงานคล้ายๆกัน แต่เราควรจะศึกษาให้เข้าใจและเลือกฟังก์ชันให้เหมาะสมกับงานของเรา



เนื้อหาแนะนำในเว็บไซต์ของเรา



ตาราง

เป็นบทที่จะอธิบายความหมายของคำว่า "ตาราง" ใน Excel และการอ้างอิง Cell เพื่อใช้ประกอบการสร้างสูตร

ฟังก์ชัน

เป็นบทที่จะรวมเนื้อหาการสร้างสูตรแบบครบวงจรและฟังก์ชันพื้นฐาน

ฟังก์ชัน VLOOKUP

เป็นบทที่จะพูดถึงฟังก์ชัน VLOOKUP ที่เป็นฟังก์ชันที่ใช้ในการค้นหาข้อมูลจากตารางที่อ้างอิง
