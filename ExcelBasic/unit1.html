เนื้อหา

Interface Table และการใส่ข้อมูล การจัดการกับ Table การดำเนินการทางคณิตศาสตร์ ตัวดำเนินการเปรียบเทียบ ฟังก์ชัน ฟังก์ชัน IF ฟังก์ชัน VLOOKUP การสร้างกราฟ เทคนิคเพิ่มเติม

โหมดกลางคืน สมาชิกกลุ่ม

ความหมายของแต่ละส่วนหน้าจอของ Microsoft Excel (Interface)

Quick Access : เราสามารถทำการเพิ่มไอคอน Tools ต่างๆที่เราใช้บ่อยๆมาไว้ตรงนี้ได้

Ribbon Bar : จะเป็นการรวม Tools ต่างๆในการจัดการ Excel

Name Box : ชื่อของ Cell นั้นๆ หรือ จะเป็นการพิมพ์ชื่อ Cell นั้นๆลงไปแล้ว Enter มันจำนำเราไปอยู่ใน Cell ที่เราต้องการ

Formula Bar : โดยเป็นช่องใส่สูตรที่เราจะใช้

Column : ชื่อคอลั่มในการใช้งานไฟล์ใน Excel อาทิเช่น A , B , C , D , E , F……

Row : ชื่อแถวในการใช้งานไฟล์ใน Excel อาทิเช่น 1,2,3,4,5

Sheet Tab : จะเป็นแถบในการจัดการ Sheet และการเพิ่ม Sheet
